Super Trunfo – Cadastro de 2 Cartas (C)
== Objetivo
Programa simples em C para cadastrar duas cartas do jogo Super Trunfo e exibir os dados no final.
== Como funciona
Solicita ao usuário os dados de cada carta:
Estado (ex.: A)
Código (ex.: A01)
Nome da cidade
População
Área (km²)
PIB (em bilhões)
Número de pontos turísticos
Armazena os dados em variáveis separadas.
Exibe todas as informações das duas cartas de forma organizada.
 ==Exemplo de execução
=== Cadastro de 2 Cartas do Super Trunfo ===
